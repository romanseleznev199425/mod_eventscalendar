# Модуль календаря событий для Joomla

Современный, адаптивный модуль календаря для Joomla 4/5 с возможностью управления событиями. Отображайте события на определённых датах с настраиваемыми заголовками и ссылками.

## ✨ Возможности

- 📅 **Современный дизайн** - Чистый, адаптивный календарь с градиентами
- 🎯 **Управление событиями** - Добавляйте неограниченное количество событий с датой, названием и ссылкой
- 📱 **Полностью адаптивный** - Отлично работает на компьютерах, планшетах и телефонах
- 🖱️ **Интерактивные всплывающие окна** - При наведении (ПК) или клике (мобильные) отображаются детали события
- 🏷️ **Счётчик событий** - Показывает количество событий на день с правильным склонением
- 🎨 **Визуальные индикаторы** - Цветовое выделение ячеек с событиями
- ⭐ **Подсветка сегодняшнего дня** - Текущая дата визуально выделена
- 🌐 **Русский язык** - Названия месяцев и дней недели на русском
- 🚀 **Лёгкий вес** - Нет внешних зависимостей, чистый CSS/JS

## 📋 Требования

## 🔧 Установка

### Способ 1: Загрузка расширения (рекомендуется)

1. Скачайте последнюю версию из @romanseleznev199425/mod_eventscalendar
2. В панели администратора Joomla перейдите в **Система → Установить → Загрузить файл пакета**
3. Нажмите **Выберите файл** и выберите скачанный ZIP
4. Нажмите **Загрузить и установить**

### Способ 2: Установка через папку

1. Распакуйте ZIP в папку `/modules/mod_eventscalendar/`
2. В панели администратора Joomla перейдите в **Система → Установить → Обнаружить**
3. Нажмите кнопку **Обнаружить**
4. Выберите модуль и нажмите **Установить**

## 🚀 Использование

### Настройка модуля

1. Перейдите в **Содержимое сайта → Модули**
2. Найдите **Календарь событий** и нажмите для редактирования
3. Установите позицию модуля и другие стандартные настройки
4. Добавьте события в разделе **Список событий**:

| Поле | Описание | Пример |
|------|----------|--------|
| Дата | Дата события (ГГГГ-ММ-ДД) | `2026-05-19` |
| Название | Название мероприятия | `Конференция 2026` |
| URL | Ссылка (полная или относительная) | `https://example.com/event` |

5. Нажмите **Сохранить**

### Добавление нескольких событий

- Нажмите **Add event** для добавления новых строк с событиями
- Нажмите на корзину для удаления событий
- События автоматически группируются по датам

### Использование на сайте

- Календарь по умолчанию отображает текущий месяц
- Даты с событиями имеют:
  - Оранжевый градиентный фон
  - Бейдж с количеством событий
  - Пульсирующую точку-индикатор
- **На компьютере**: Наведите курсор на дату для просмотра деталей события
- **На мобильных**: Нажмите на дату для просмотра деталей в нижнем всплывающем окне

## 🎨 Настройка внешнего вида

🔧 Устранение неполадок
Календарь не отображается
Убедитесь, что модуль опубликован на правильную позицию

Проверьте, что в файле index.php вашего шаблона есть <?php echo $this->getBuffer('modules', 'позиция') ?>

События не отображаются
Проверьте формат даты: ГГГГ-ММ-ДД (пример: 2026-05-19)

Очистите кеш Joomla: Система → Очистить кеш

Всплывающее окно не работает
Проверьте консоль браузера на наличие JavaScript ошибок

Убедитесь, что нет конфликтов с другими скриптами

CSS не загружается
Проверьте права доступа к файлам: 644 для CSS файлов

Очистите кеш браузера и кеш Joomla

### Переопределение стилей CSS

Добавьте в CSS вашего шаблона:

```css
/* Изменение цветовой схемы календаря */
.mod-events-calendar .has-event > div {
    background: linear-gradient(135deg, #ваш-цвет-1, #ваш-цвет-2);
}

/* Изменение цвета бейджа с событиями */
.mod-events-calendar .events-count-number {
    color: #ваш-цвет;
}

/* Изменение цвета сегодняшнего дня */
.mod-events-calendar .calendar-cell.today > div {
    background: linear-gradient(135deg, #ваш-градиент);
}

/* Изменение цвета дней недели */
.mod-events-calendar .weekday th {
    color: #ваш-цвет;
    background: #ваш-фон;
